# Conversor de Unidades

Proyecto simple para convertir unidades de temperatura, distancia y peso.

## Estructura

- `index.html`: interfaz de usuario.
- `styles.css`: estilos modernos y responsivos.
- `script.js`: lógica de conversión para todas las unidades.

## Cómo usar

1. Abre `index.html` en el navegador.
2. Ingresa un valor numérico.
3. Elige la categoría, unidad de origen y unidad de destino.
4. Haz clic en `Convertir`.

## Conversiones incluidas

- Temperatura: Celsius ↔ Fahrenheit ↔ Kelvin
- Distancia: Metros ↔ Kilómetros ↔ Millas
- Peso: Gramos ↔ Kilogramos ↔ Libras
